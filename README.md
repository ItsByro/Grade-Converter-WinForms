Another project, this time it is a Grade Converter

Grade Converter - WinForm App
- A windows form application for checking grades such as GPA or GWA.

## Features:
- GWA CONVERSION: General Weighted Average for Philippine DepEd Education Including:
  - High School / Senior High School (DepEd Format)
  - College (Weighted Average)
- GPA CONVERSION: Weighted Average Calculator (For International ><)
- Slick and easy to navigate menu
- some hidden easter egg idk?:p

## Build with:
- C# (v4.5.1)
- SharpDevelop IDE
- Windows Forms
